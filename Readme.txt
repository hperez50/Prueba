
Para el desarrollo de la aplicacion utilice lo siguiente, (aclaro que no termine la aplicacion por disponibilidad de tiempo, pero presento lo que alcance a desarrollar).

1) Configure la base de datos en mysql.
2) Conectarse a la base de datos desde el servidor de aplicaciones:
   a. Crear recurso JDBC
   b. Crear Connection Pool
3) Crear proyecto
    a. Crear la clase modelo con anotaciones JPA.
    b. Crear componentes DAO(EJB).
4) Crear la unidad de persistencia.
5) Crear el componente de controlador(Servlet).
6) Crear la pagina web para el modelo vista(JSP).
